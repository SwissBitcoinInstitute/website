# Simplified Payment Verification (SPV)

Simplified Payment Verification (SPV) is a method allowing light nodes to verify transactions without downloading the full blockchain. By checking block headers and Merkle proofs, an SPV client can confirm that a transaction is included in a valid block, though it trusts that the majority of miners are following the Bitcoin consensus rules
