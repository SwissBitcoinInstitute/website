# Mainnet

Mainnet refers to the operational Bitcoin network where real Bitcoin transactions take place. It is distinguished from testnet, which is a parallel environment for experimentation without economic value. Protocol changes and software upgrades are deployed cautiously on mainnet to avoid disruption.
